# E-Commerce Store

A minimal e-commerce storefront built with React, Vite, and TailwindCSS.

## Tech Stack
- **Frontend**: React 18 + Vite
- **Styling**: TailwindCSS
- **Icons**: Lucide React

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Dev server runs on [http://localhost:3088](http://localhost:3088).

## Features
- Product listing grid
- Add to cart / remove from cart
- Cart sidebar with quantity controls
- Responsive design
